Riots.im Privacy Notice

This site provides access to hosted instances of historic releases of Riot Web.

Each Riot Web instance is pulled direclty from GitHub, and deployed to S3 using the default configuration proivided in the GitHub release tar. This means that the configuration is provided by New Vector Ltd., and is likely to use New Vector defaults for services including: homeserver, identity server, integration manager, opt-in analytics, debug log target, and welcome bot. For more information on how New Vector Ltd will process your data, see the information on matrix.org and vector.im.

The site is hosted using Amazon's S3, CloudFront, Route 53 and Lambda@Edge. No personal data is logged during the serving of this site to you. For more information, please see Amazon's AWS privacy notice.
